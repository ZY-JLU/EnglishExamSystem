# EnglishExamSystem
英语考试系统。主要包括模拟考试、对考题进行分析，根据四级题型细分为听力、阅读、翻译和写作四个小模块。APP的所有功能将紧紧围绕着这四个模块进行设计，具体功能如下：考试、考试成绩结算、错题标记、习题收藏、常考词汇查询等。
